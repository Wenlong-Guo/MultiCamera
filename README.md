## MultiCamera

* 修复了的已知Bug
  1. 预览比例和GLSurfaceView比例一致 多余切割掉 类似CenterCrop
  2. 切换前后摄像头颠倒一帧
  3. 多个多个GLSurfaceView 会黑屏 (需手动调用API处理)
  4. 切换摄像头 不卡 主线程
  
* 未修复的Bug
  1. 暴力切换两个Fragment的相机预览 由于相机操作是耗时任务 会导致卡顿 (背压问题)
  2. 手动对焦不可用
